## 🎓 技術實作細節

### 標籤平滑數學
```
硬目標：   P(正確) = 1.0, P(其他) = 0.0
平滑目標： P(正確) = 0.9, P(其他) = 0.1/(V-1)

損失 = -Σ P_smooth(y) * log(P_model(y))
```

### 學習率排程
```python
lr = d_model^(-0.5) * min(
    step^(-0.5), 
    step * warmup_steps^(-1.5)
)
```

視覺化：
```
LR |    ╱────╲_____
   |   ╱          ╲____
   |  ╱                ╲____
   | ╱                      ╲____
   └──────────────────────────────→ 步數
     預熱  峰值    衰減
```

### 遮罩策略

**填充遮罩**（忽略 <pad> 詞元）：
```
序列：[5, 8, 3, 0, 0]  (0 = 填充)
遮罩：[1, 1, 1, 0, 0]  (1 = 注意，0 = 忽略)
```

**因果遮罩**（防止看到未來）：
```
[[1, 0, 0, 0],   # 位置 0：只看位置 0
 [1, 1, 0, 0],   # 位置 1：看位置 0-1
 [1, 1, 1, 0],   # 位置 2：看位置 0-2
 [1, 1, 1, 1]]   # 位置 3：看所有位置
```

**組合遮罩** = 填充 AND 因果

## 📊 訓練流程

```
┌─────────────────────────────────────────────────────────────┐
│                      訓練流程                                │
└─────────────────────────────────────────────────────────────┘

1. 資料載入
   ├─ 生成序列（複製/反轉/排序）
   ├─ 加入特殊詞元（<start>、<end>）
   ├─ 建立帶填充的批次
   └─ 分割訓練/驗證（90%/10%）

2. 模型初始化
   ├─ 建立 Transformer（encoder + decoder）
   ├─ Xavier 初始化
   └─ 移動到裝置（CPU/CUDA）

3. 訓練設置
   ├─ Adam 優化器（β1=0.9, β2=0.98）
   ├─ Transformer 學習率排程器
   ├─ 標籤平滑損失
   └─ 梯度裁剪

4. 訓練迴圈（每個 epoch）
   ├─ 前向傳播
   │  ├─ 建立遮罩（填充 + 因果）
   │  ├─ 編碼源序列
   │  ├─ 解碼目標序列（教師強迫）
   │  └─ 投影到詞彙表
   │
   ├─ 計算損失
   │  ├─ 標籤平滑
   │  └─ 忽略填充位置
   │
   ├─ 反向傳播
   │  ├─ 計算梯度
   │  ├─ 裁剪梯度（max_norm=1.0）
   │  └─ 更新權重
   │
   ├─ 更新學習率
   │  └─ 預熱 → 峰值 → 衰減
   │
   └─ 追蹤指標
      ├─ 損失
      ├─ 詞元準確率
      └─ 序列準確率

5. 驗證
   ├─ 不計算梯度
   ├─ 相同前向傳播
   └─ 計算指標

6. 檢查點
   ├─ 儲存最佳模型（最高驗證準確率）
   ├─ 每 N 個 epoch 儲存
   └─ 儲存最新（用於恢復）

7. 生成測試（每 5 個 epoch）
   ├─ 自回歸生成
   ├─ 顯示 5 個範例
   └─ 與預期輸出比較
```
